# Watermark_Generator
快速生成简单的水印，作为批量渲染、媒体编码器或编辑时间线的叠加。

---

中文 | [English](./README_en.md) 

## 使用依赖
```console
pip install pillow
```

## 如何使用
#### 方法一
```console
python Watermark_Generator.py
```
在水印文本上附加当前日期
```console
python -d Watermark_Generator.py
```
